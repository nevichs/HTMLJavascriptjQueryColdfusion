In this repository you will find a few code examples that utilize a combination of HTML, Javascript, Jquery, and Coldfusion.

TREE GRID
This is an example of a jQuery tree grid with extra features added including node saving and extra search features

Requires
-jQuery framework
-jQuery UI (for tabs and grid)


EMAIL SCREEN
THis is an example of an email screen that uses a javascript object to contain the email recipients. It allows for sending an email with refreshing the screen.

Requires
-jQuery framework
-jQuery UI (for tabs)

KEY WORD HIGHLIGHTING
This shows an example of highlighting a certain keyword in the results and then requering the results to reorder it based upon relevance.

Notes
-Completely coldfusion code.
-Uses a "<font>" tag to highlight.
